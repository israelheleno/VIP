# ✈️ VIP Pass Tracker

O **VIP Pass Tracker** é um Web App progressivo (PWA) desenvolvido para viajantes que possuem múltiplos cartões de crédito de alta renda e precisam de uma forma rápida, visual e offline de controlar seus acessos a salas VIP.

## ✨ Funcionalidades
- **Banco de Dados Atualizado (2026):** Inclui regras de mais de 40 cartões, desde variantes Black/Infinite até os exclusivos segmentos Private (Centurion, DUX, Aeternum, etc.).
- **Controle de Saldo:** Monitore quantos acessos você já utilizou e quanto ainda tem de saldo em tempo real.
- **Lógica de Adicionais:** Informações claras sobre se o benefício é compartilhado ou individual para cada cartão.
- **Funciona Offline:** Uma vez aberto, o app salva os dados localmente no seu dispositivo (LocalStorage), permitindo consulta mesmo em modo avião.
- **Privacidade Total:** Seus dados de uso ficam salvos apenas no seu próprio navegador; nada é enviado para servidores externos.
- **Modo de Compartilhamento:** Interface otimizada para capturas de tela (Print) para envio rápido de status de acessos.

## 🚀 Como usar
1. Adicione seus cartões através do menu de seleção.
2. Use os botões `+` e `-` para atualizar seus acessos conforme for utilizando.
3. No iPhone ou Android, use a função **"Adicionar à Tela de Início"** para transformar o site em um App com ícone personalizado.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Design Responsivo & Dark Mode)
- JavaScript Vanilla (Lógica de contagem e persistência)
- GitHub Pages (Hospedagem)
